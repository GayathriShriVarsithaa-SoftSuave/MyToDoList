# 📝 MyToDoList App (Android - Kotlin)

A simple To-Do List Android application built using Kotlin, Material Components, and MVVM architecture.

---

## 🚀 Features Implemented So Far
- Created three main screens:
  - **Home** – View and manage tasks.
  - **Search** – Search for tasks.
  - **Detail** – View task details.
- Used **Material Components** for modern UI.
- Implemented MVVM architecture
- Fragment-based navigation implemented.

---

## 🔮 Future Improvements
- Add persistent storage (Room Database).
- Implement cloud sync.
- Task tagging and Many-to-Many relationships. 
- Advanced search and filtering.
